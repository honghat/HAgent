Kết nối tài khoản ChatGPT

1️⃣ Lấy access token từ ChatGPT

  1. Mở tab chatgpt.com trong Chrome, đã đăng nhập
  2. F12 → Console
  3. Paste: copy(localStorage.getItem('accessToken')) và Enter
  4. Nếu thấy thông báo "Copied", token đã vào clipboard

2️⃣ Dán token vào form

Nhập email gợi nhớ (VD: hat.hopnhat2) và token vào ô dưới:

Sau khi kết nối, token được lưu vào tokens/chatgpt2api_accounts.json.

Token không bao giờ gửi ra ngoài — chỉ dùng gọi ChatGPT API locally.
